按字母顺序排列。
## Array.from()
从一个类似数组或可迭代对象创建一个新的，浅拷贝的数组实例。
#### demo
{% highlight js%}
console.log(Array.from('foo'));
// expected output: Array ["f", "o", "o"]

console.log(Array.from([1, 2, 3], x => x + x));
// expected output: Array [2, 4, 6]
{% endhighlight %}
#### 语法
{% highlight js %}
Array.from(arrayLike[, mapFn[, thisArg]])
{% endhighlight %}
#### 参数
* arrayLike
源数组
* mapFn(可选)
每个元素都会执行的回调函数
* thisArg(可选)
执行mapFn时的this对象

#### 返回值
**新的**数组实例

## Array.every()
测试一个数组内的所有元素是否都能通过某个指定函数的测试。它返回一个布尔值, **不改变源数组**。
#### demo
{% highlight js %}
const isBelowThreshold = (currentValue) => currentValue < 40;

const array1 = [1, 30, 39, 29, 10, 13];

console.log(array1.every(isBelowThreshold));
// expected output: true
{% endhighlight %}
#### 语法
{% highlight js %}
arr.every(callback[, thisArg])
{% endhighlight %}
#### 参数
1. callback 接收3个参数
   1. element 数组元素
   2. index 索引
   3. array 调用every的源数组
2. thisArg

#### 返回值
Boolean.

## Array.filter()
创建一个新数组, 其包含通过所提供函数实现的测试的所有元素。
#### demo
{% highlight js %}
const words = ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present'];

const result = words.filter(word => word.length > 6);

console.log(result);
// expected output: Array ["exuberant", "destruction", "present"]
{% endhighlight  %}
#### 语法
{% highlight js %}
var newArray = arr.filter(callback(element[, index[, array]])[, thisArg])
{% endhighlight %}
#### 参数
1. callback 返回 `true` 表示该元素通过测试，保留该元素，`false` 则不保留。接收3个参数：
   1. element 数组元素
   2. index 索引
   3. array 调用filter的源数组
2. thisArg

#### 返回值
一个**新的**、由通过测试的元素组成的数组，如果没有任何数组元素通过测试，则返回空数组

## Array.find()&Array.findIndex()
返回数组中满足提供的测试函数的**第一个元素**的值(或索引)。
#### demo
{% highlight js %}
const array1 = [5, 12, 8, 130, 44];

const found = array1.find(element => element > 10);

console.log(found);
// expected output: 12
{% endhighlight %}
#### 语法
{% highlight js %}
arr.find(callback[, thisArg])
{% endhighlight %}
#### 参数
1. callback 接收3个参数
   1. element 数组元素
   2. index 索引
   3. array 调用every的源数组
2. thisArg

#### 返回值
数组中**第一个**满足所提供测试函数的元素的值（或索引），否则返回 undefined（或-1）。
